# Brno útiterv

Egy statikus HTML útiterv oldal Budapest → Brno őszi utazáshoz.

## Helyi előnézet

Nyisd meg az `index.html` fájlt egy böngészőben.

## GitHub Pages publikálás

1. Pushold a repositoryt GitHubra.
2. Menj a repository `Settings` → `Pages` oldalára.
3. `Source`: `Deploy from a branch`.
4. `Branch`: `main`.
5. `Folder`: `/root`.
6. `Save`.

## Várható URL-formátum

```text
https://USERNAME.github.io/REPOSITORY-NAME/
```

Megjegyzés: a GitHub Pages deployment néhány percet igénybe vehet.
